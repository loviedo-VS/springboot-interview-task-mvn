# Spring Boot Interview Exercise

### Task
Create a simple REST API that:
- `GET /users` → returns a list of users
- `POST /users` → adds a new user (in-memory list is fine)
- `GET /users/{id}` → returns a user by ID

### Requirements
- Use a plain Java list or map for data storage
- Add at least one JUnit test
- Use Spring Boot (no database needed)
